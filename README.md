# BetterGovernors Mod for Bannerlord

Welcome to the **BetterGovernors** mod for Mount & Blade II: Bannerlord! This mod enhances the functionality of governors in your game, providing a more immersive and strategic experience.

## Installation

Follow these steps to install the **BetterGovernors** mod:

1. **Download the Mod**:
    - Download the latest version of **BetterGovernors.zip** from the releases section.

2. **Locating the Modules Folder**:
    - Navigate to your Bannerlord installation directory. Typically, this can be found at `C:\Program Files (x86)\Steam\steamapps\common\Mount & Blade II Bannerlord`.

3. **Installing the Mod**:
    - Unzip `BetterGovernors.zip`. Copy the entirety of the now unzipped `BetterGovernors` folder (which contains the compiled `.dll` file) into the `Modules` folder of your Bannerlord installation.
    
    For example, your directory structure should look something like this:
    ```
    Mount & Blade II Bannerlord
    └── Modules
        └── BetterGovernors
            └── [files from the mod]
    ```

4. **Activating the Mod**:
    - Launch Bannerlord, and in the launcher, go to the 'Mods' tab.
    - Find **BetterGovernors** in the list and check the box next to it to activate the mod.

5. **Play**:
    - After activating the mod, start the game. **BetterGovernors** will now be enhancing your Bannerlord experience!

## Features

- **Governor Enhancement**: Governors resolve settlement issues. Governors will gain xp in skills with governor perks every day, with a higher chance of gaining xp in a skill associated with governing, and a lower chance for skills such as combat.

- **Configurable Settings**: (WIP) Customizable settings to tailor the governor's skill gain to your preference.

- **Daily Updates**: The mod performs checks and updates on each in-game day, ensuring that the governors' actions and effects are consistently applied.

## Compatibility

- **Game Version**: Designed for Mount & Blade II: Bannerlord v1.2.9.

- **Mod Compatibility**: Should be compatible with most other mods, but conflicts may occur with mods that significantly alter governor mechanics.
